# GITA PYQ Repository (JSON Version)

A simple Flask-based web app to manage and browse **GITA Autonomous College previous year question papers**.  
This version uses a JSON file (`papers.json`) for storing metadata instead of a database, making it lightweight and beginner-friendly.
